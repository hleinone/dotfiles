# dotfiles

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install zero-sh/tap/zero
git clone --recursive https://github.com/hleinone/dotfiles.git ~/.dotfiles
zero setup
```
